# React + Typescript + React Hook Form + Zustand + TailwindCSS

```
npm install tailwindcss @tailwindcss/vite
```
### React Hook Form
```
npm i react-hook-form
```
### Zustand
```
npm i zustand
```

```
npm i uuid
npm i --save-dev @types/uuid
```

```
npm i react-toastify
```
